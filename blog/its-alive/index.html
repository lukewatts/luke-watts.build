It's Alive!

August 03 2022

I'm back! I've been wanting to re-launch my blog for a long time now, and so when I heard about Astro I knew it was time.

I'm still working on some things, but I'm happy with the progress so far. As I mentioned, I'm using Astro, which is a new static site generator that uses React components (or Vue, Svelte, Alpine etc). I'm also using Tailwind CSS for styling. I'll probably use AlpineJS as well, because I really like it! Goes really well with React and Tailwind. So for now I'm just using the default blog template from Astro CLI, but I will be redesigning the site from the ground up over the next few weeks.

Where have I been?

So I had a fairly popular blog back in 2014-2017. Popular enough that Packt Publshing reached out to me to write a book in 2015. I wrote a book called Mastering Sass which was published in August 2016. That was back when I worked for myself. I had started a small web design agency a few years before, and my blog was about new technologies I was learning so I (we/Affinity4) could compete in the market, and Sass (& Compass) was one of them.

Then, in 2017 I took a job with the leading Telecommunications company in Ireland, Three Ireland. I stopped blogging and accidentally let my domain expire for a week, in which time some Chinese scammer bought it and filled created a ad-filled garbage pile of a website. After a few years I guess they got bored of it and I managed to buy the domain again last at the start of the pandemic (2019). I knew I needed to wait for the bad rep my domain had to die down before I could start blogging again. So I just left it offline since then.

Why am I back?

So while working with Three, I had a lot of life happen. I travelled a lot, got married, bought a house, and most recently started a new job as a Senior Developer with a successful startup called Workvivo.

So with that I'm learning a lot of new things, and re-learning some forgotten things. So I thought it was time to start blogging again, and sharing what I'm learning, like I used to do.

What's next?

I'm going to be writing about a lot of things, but I'm going to try and focus on the following:

  • I'm building a full-stack framework called Noesis, which uses InertiaJS, React, Tailwind, and my own PHP MVC framework called Dianoia. I'm going to be writing about that a lot.
  • Laravel
  • React
  • AlpineJS
  • AWS
  • And probably a bit about Astro as well, as I use it to rebuild this site, and maybe others, like the Noesis/Dianoia docs.

Conclusion

So that's it for now. I'm going to be writing a lot more, so stay tuned!
